# A vagal liver brain circuit for inflammation sensing and response
This repository contains the code used to generate some of the figures in the manuscript using R.
In this manuscript, Fig. 1j–n, Fig. 2d–g, Fig. 5d–g, Extended Data Fig. S2a–s, Extended Data Fig. S8a–f, Extended Data Fig. S10a–c, and Extended Data Fig. 12f, i, l, o–q, s–u were generated using R. The code is organized by figure number.
